# Sweet Treats PH Jobscout

A web platform for Sweet Treats PH, a Filipino dessert restaurant, to connect job seekers and restaurant management. Users can create profiles, upload resumes, apply for jobs, and communicate with admins, while admins manage job postings and staff efficiently.

## About
Sweet Treats PH Job Portal is designed to streamline hiring for the dessert restaurant. Job seekers can manage profiles, upload resumes, apply for positions, and communicate directly with restaurant admins. Admins can create job openings, manage staff accounts, and respond to applicants through a built-in messaging system.

## Features
- **User Role**
  - Register and login
  - Customize profile
  - Upload resume/CV
  - Apply for restaurant jobs
  - Send and receive messages with Admin
- **Admin Role**
  - Create and manage job postings
  - Manage staff accounts manually
  - Send and receive messages with applicants
  - Access full dashboard for oversight
  - Analytics and Reports
  - Review applicants

## Technologies Used
- Vue.js 3, Bootstrap 5, Inertia.js
- Laravel 11
- MySQL
- Composer, NPM
