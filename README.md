# Sweet Treats PH JobScout

A web platform for job seekers and employers in the Filipino dessert industry. Users can create profiles, upload resumes, apply for jobs, and communicate with admins, while admins manage job postings and members efficiently.

## About
Sweet Treats PH JobScout simplifies the recruitment process for dessert businesses. Users can manage their profiles, upload resumes, and apply for jobs with customized interview questions. Admins can create jobs, manage members, and communicate with applicants through a built-in messaging system.

## Features
- **User Role**
  - Register and login
  - Customize profile
  - Upload resume/CV
  - Apply to jobs with interview questions
  - Send and receive messages with Admin
- **Admin Role**
  - Create and manage jobs
  - Create and manage members manually
  - Send and receive messages with Applicants
  - Access full dashboard for oversight

## Technologies Used
- Vue.js 3, Bootstrap 5, Inertia.js
- Laravel 11
- MySQL
- Composer, NPM
